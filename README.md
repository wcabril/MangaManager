# MangaManager 📚

A desktop application to organize manga files automatically, including metadata fetching and chapter management.

## 🚀 Features

- 📦 Extract and organize CBZ files
- 🏷️ Automatic metadata detection
- 👤 Author lookup via AniList
- 🔄 Mangadex fallback for missing data
- 🧹 Clean and move manga files into structured folders

## 🛠️ Tech Stack

- C#
- .NET
- WPF
